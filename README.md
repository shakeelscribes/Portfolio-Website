## 🌌 The Architecture

A modern, highly interactive personal portfolio engineered to showcase technical depth. Built from the ground up utilizing a bespoke glassmorphism design system to emulate the sleek, precise feel of a modern terminal environment.

> **[🚀 INITIALIZE LIVE DEMO](https://portfolio-website-shakeelscribes.vercel.app/)** *(Update with Vercel URL)*

## ✨ System Features

- **Next.js 14+ (App Router):** Fast, SEO-friendly, and optimized edge delivery.
- **Glassmorphism Design System:** Custom CSS variables injecting a sleek, translucent, deep-space aesthetic.
- **Fluid Typography:** Typography scales mathematically across all device viewports.
- **Interactive Animations:** Powered by Lenis (Smooth Scrolling) and Framer Motion for scroll reveals and hyper-customized hover states.
- **Terminal Stacking Cards:** Projects presented in an overlapping, interactive console-style interface.
- **Direct Slack Integration:** Contact form submissions bypass traditional email relays, securely routing directly into a dedicated Slack workspace via custom API routes.
- **Supabase Backend:** Securely logs and encrypts all incoming transmissions.

## 💻 Local Deployment Protocol

To initialize this environment locally, execute the following sequences:

**1. Clone the repository:**
```bash
git clone https://github.com/shakeelscribes/Portfolio-Website.git
cd Portfolio-Website
```

**2. Install dependencies:**
```bash
npm install
```

**3. Configure Environment Variables:**
Rename `.env.local.example` to `.env.local` and inject your keys:
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SLACK_WEBHOOK_URL=your_slack_webhook_url
```

**4. Boot the server:**
```bash
npm run dev
```
Access the local port at [http://localhost:3000](http://localhost:3000).
